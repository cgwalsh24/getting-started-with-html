How the Internet works

People want to access web pages, so they use...

Web browsers - programs like Internet Explorer, Chrome, and Firefox that display websites on user's screens and translate...
HTML (Hyper Text Markup Language) - this is code that computers use to structure and interpret content on web pages.

Through web browsers people request to see specific websites like www.udacity.com, and send an HTTP request across the INTERNET to connect with...
SERVERS - powerful computers designed to store websites, interpret http requests, and send that data at lightning speeds.

HTML

As mentioned earlier HTML stand for Hyper Text Markup Language.

It is the code we use to structure websites, and it's needed because computers are dumb and need a specific, more technical language to converse with one another.

Tags and Elements

Each website has at least one HTML document, most often multiple documents that contain ELEMENTS. These are distinct ways in which we can highlight, structure, and design content on the website. We do so by using TAGS.

Tags have diverse functions and are employed in a variety of ways.

    Some of the tags we were introduced to include...
  • (a) tags - which link content to other sites across the web
  • (img) tags - which include their own reference sources and displays images on websites
  • (em) tags - which "em"phasize text
  • (div) tags which delinate space on the site
  • In addition to a number of others
Each tag can either be classified as...
-inline
-block
